Mis Proyectos

Portafolio de Desarrollo de Software

Aquí están algunos de mis proyectos más significativos que muestran mis habilidades en desarrollo de software, aplicaciones en tiempo real y diseño de experiencia de usuario.

Plataforma de Subastas Online en Vivo

Aplicación Web en Tiempo Real

Una plataforma integral de subastas online donde los usuarios pueden pujar en tiempo real por diversos artículos incluyendo arte, joyería y coleccionables. El sistema incluye transmisión en vivo de subastas, actualizaciones de pujas en tiempo real y procesamiento seguro de pagos.

Características Principales:

  • Sistema de pujas en tiempo real con actualizaciones en vivo
  • Transmisión en vivo y difusión de subastas
  • Autenticación segura de usuarios y procesamiento de pagos
  • Catálogo de artículos con descripciones detalladas e imágenes
  • Historial de pujas y análisis de subastas
  • Diseño responsive para móviles

Tecnologías Utilizadas:

JavaScript Node.js WebSocket React Mysql Express Socket.io

Integración VTEX - Ecomexperts

Integración de E-commerce

Sistema de integración entre VTEX y Ecomexperts para sincronizar precios y stock de productos, además de traer datos de ventas en tiempo real. Solución que automatiza la gestión de inventario y optimiza el flujo de información entre plataformas.

Características Principales:

  • Sincronización automática de precios entre plataformas
  • Gestión de stock en tiempo real
  • Importación de datos de ventas
  • Automatización de procesos de inventario
  • Procesamiento de notificaciones en AWS
  • Workers para tareas asíncronas
  • Monitoreo de integridad de datos
  • Reportes de sincronización y errores

Tecnologías Utilizadas:

VTEX API PHP JavaScript REST API MySQL AWS

Plataforma Omnicanal de Gestión Comercial

Automatización de Canales de Venta

Desarrollo y mantenimiento de un sistema que centraliza canales de venta, sincroniza catálogos y automatiza la emisión de facturas electrónicas para múltiples países, garantizando operaciones 24/7.

Características Principales:

  • Sincronización de stock y precios con marketplaces y tiendas propias
  • Motor de facturación electrónica para Argentina,Chile y México
  • Dashboard de ventas con analítica en tiempo real
  • Orquestación de jobs y colas para procesos críticos
  • Alertas automáticas ante inconsistencias de inventario

Tecnologías Utilizadas:

Node.js TypeScript PostgreSQL Redis AWS REST API
